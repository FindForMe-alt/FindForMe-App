# FindForMe Starter App

A working starter web app for a reverse marketplace where users post "In Search Of" requests and other users or businesses submit offers.

## Features included
- In Search Of request feed
- Item and Labor / Odd Job categories
- Business page starter concept
- Submit offers / bids
- Requester accepts offer before chat opens
- Sourcer can pass, which moves request to Open Again
- Phone-friendly responsive layout

## Run it
1. Install Node.js
2. Open this folder in Terminal
3. Run:

```bash
npm install
npm run dev
```

Then open the local URL Vite gives you.
